## üîç How It Works

The Vite RSC Browser Example is designed to demonstrate a specific method for using React Router RSC without relying on a server. It utilizes a unique way for components to run directly in the browser, streamlining the process and enhancing performance.

React Router RSC provides an API called `fetch`. By overriding this API, the application ensures all RSC requests are handled effectively within your browser.
